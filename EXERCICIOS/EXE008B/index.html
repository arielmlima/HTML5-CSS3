Código-Fonte / Pré-Formatação

A tag CODE é responsavel por formatar a fonte com Monoespaçamento. Oferecendo um tamanho e distancia específicas remetendo ao código de computador

A tag PRE é responsavel por manter a identação exatamente igual está escrita no código, dando liberdade ao usuario para quebrar linhas e pular parágrafos com espaços e enter.
